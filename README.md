# MarketHub - Buy & Sell Used Products

A simple, effective platform for buying and selling used products with a clean, intuitive interface.

## Framework

Vanilla JavaScript with Tailwind CSS (CDN)

## Features

- **Browse Products**: Search and filter through available listings
- **Category Filtering**: Electronics, Furniture, Clothing, Books, Sports & Outdoors
- **Product Details**: View full product information with seller contact
- **List Items**: Easy-to-use form for selling your used products
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop
- **Real-time Search**: Instant filtering as you type

## Preview

Simply open `index.html` in your browser to view the application.

## Theme

Clean Light theme with:
- White and soft gray backgrounds
- Blue accents for primary actions
- Subtle borders and shadows
- Professional, accessible design
- Calm, minimal aesthetic

## Usage

1. **Browse Products**: Use the search bar and category filter to find items
2. **View Details**: Click on any product card to see full information
3. **Contact Seller**: Use the contact button to reach out via email
4. **List Your Item**: Click "+ List Item" to add your product for sale

## Development

No build process required. Just open `index.html` in a web browser.

All product data is stored in memory for demonstration purposes.
